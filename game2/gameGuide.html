Guide du Jeu - Memory Card Game
Bienvenue dans le Memory Card Game!
L'objectif est de trouver toutes
les paires de cartes correspondantes.
Amusez-vous à jouer seul
ou défiez un ami en mode deux joueurs!

Suivez les étapes ci-dessous pour jouer.

Démarrer le Jeu

Ouvrez le jeu en cliquant sur "Start Game".

Choisissez le nombre de joueurs (1 ou 2) en cliquant sur les boutons radio.

Étape 1

:Début du jeu
Au début de la partie, toutes les cartes sont mélangées et disposées en rangées, face cachée sur la table.

Étape 2

Le joueur commence et retourne deux cartes

Étape 3

Dans ce cas, les cartes ne correspondent pas,
ce n’est pas une paire,
il les retourne donc.

Étape 4

Si les cartes correspondent, elles resteront retournées face visible.

Étape 5

Mémorisez l'emplacement des cartes pour minimiser les erreurs.
Essayez de terminer le jeu avec le moins d'erreurs possible.
Le jeu se termine lorsque toutes les paires sont trouvées.
Félicitations!
Un message de victoire s'affiche avec le nombre de tentatives,
de fautes et le temps écoulé.
